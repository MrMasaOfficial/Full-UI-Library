Getting Started

To use the UI Library, simply include the CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="css/style.css"> <script src="js/main.js"></script>

All components are ready to use with minimal configuration.

Buttons

Buttons are used to trigger actions or navigate within your application.

Basic Button

<button class="btn btn-primary">Click Me</button>

Button Variants

Available color variants:

  • btn-primary - Blue (primary action)
  • btn-secondary - Gray (secondary action)
  • btn-success - Green (positive action)
  • btn-danger - Red (destructive action)
  • btn-warning - Yellow (cautionary action)
  • btn-info - Cyan (informational action)

Button Sizes

Class Description
btn-sm Small button
(default) Medium button
btn-lg Large button

Button Styles

Use btn-outline for outlined buttons:

<button class="btn btn-primary btn-outline">Outline Button</button>

Disabled State

<button class="btn btn-primary" disabled>Disabled</button>

Cards

Cards are containers for displaying grouped content with optional headers, images, and footers.

Basic Card

<div class="card"> <div class="card-header">Card Title</div> <div class="card-body"> <p>Card content goes here</p> </div> <div class="card-footer"> <button class="btn btn-primary btn-sm">Action</button> </div> </div>

Card with Image

<div class="card"> <div class="card-header">Card Title</div> <div class="card-image"> <img src="image.jpg" alt="Description"> </div> <div class="card-body">Content</div> </div>

Card Variants

Class Description
card Basic card with default shadow
card-raised Card with elevated shadow

Card Parts

  • card-header - Top section, great for titles
  • card-image - Image container
  • card-body - Main content area
  • card-footer - Bottom section, typically for actions

Modals

Modals are dialog boxes that overlay the main content and require user interaction.

Modal Markup

<div id="myModal" class="modal"> <div class="modal-content"> <div class="modal-header"> <h2>Modal Title</h2> <button class="modal-close" onclick="closeModal('myModal')">×</button> </div> <div class="modal-body"> <p>Modal content</p> </div> <div class="modal-footer"> <button class="btn btn-secondary" onclick="closeModal('myModal')">Close</button> <button class="btn btn-primary">Save</button> </div> </div> </div>

Opening and Closing Modals

<button class="btn btn-primary" onclick="openModal('myModal')">Open Modal</button> <button onclick="closeModal('myModal')">Close</button>

Modal Sizes

Class Max Width
(default) 600px
modal-sm 400px

Closing Modals

  • Click the close button (√ó)
  • Click outside the modal
  • Press the ESC key
  • Call closeModal(modalId)

Grid System

A simple CSS Grid-based layout system for creating responsive layouts.

Two Column Grid

<div class="grid grid-2"> <div class="grid-item">Column 1</div> <div class="grid-item">Column 2</div> </div>

Grid Classes

Class Columns (Desktop) Columns (Tablet)
grid-2 2 1
grid-3 3 1
grid-4 4 2

Responsive Behavior

  • grid-2 and grid-3 collapse to 1 column on screens below 480px
  • grid-4 becomes 2 columns below 768px
  • All grids use 1.5rem gap between items

CSS Variables

All colors are defined using CSS custom properties that change with the theme:

--primary-color: #007bff --text-dark: #212529 --bg-white: #ffffff --border-color: #dee2e6

Customizing Colors

Override CSS variables in your own stylesheet:

:root { --primary-color: #ff6b6b; --success-color: #51cf66; } [data-theme="dark"] { --text-dark: #e9ecef; }
